# Simple and fast MarkDown editor in Rust

## Build

Dioxus deps
```
sudo apt update
sudo apt install libwebkit2gtk-4.1-dev \
  build-essential \
  curl \
  wget \
  file \
  libxdo-dev \
  libssl-dev \
  libayatana-appindicator3-dev \
  librsvg2-dev

cargo install dioxus-cli
```

## Test

```
cargo run -- README.md
```
